# System Performance Analysis and Fault Detection in Python

This project simulates time-series system performance data and applies basic fault detection methods to identify abnormal behavior.

## Features
- Simulates noisy system data
- Injects fault conditions such as spikes and drops
- Applies moving average smoothing
- Detects anomalies using residual thresholding
- Visualizes normal behavior and detected faults

## Tools Used
- Python
- NumPy
- Matplotlib

## Purpose
This project demonstrates how data analysis techniques can be used to monitor system health and detect faults in engineering applications.
